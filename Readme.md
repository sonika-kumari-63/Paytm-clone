<h1>Paytm Clone – HTML & CSS</h1>  
A simple frontend Paytm Clone built using HTML & CSS. This project recreates the look and feel of Paytm's homepage, including navigation, banners, sections, and footer.
A fully responsive Paytm Homepage Clone built using pure HTML and CSS.
It is designed to help understand real-world layout techniques and front-end structuring without using any JavaScript or frameworks.

<h2>Project Overview</h2>  
This Paytm clone aims to replicate the clean, user-friendly design of Paytm's homepage.
The layout includes:
 .Navbar with logo and menu links
 .Hero banner section with CTAs
 .Recharge, Pay Bills & Bookings section
 .Paytm Wallet & Payments Bank cards
 .Promotional sections & feature highlights
 .Footer with multiple columns

The entire UI is built from scratch using HTML & CSS only — no libraries, no frameworks.
